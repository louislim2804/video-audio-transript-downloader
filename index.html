5 What sites work支持哪些网站

It uses yt-dlp under the hood, which supports 1000+ sites. In practice: 底层使用 yt-dlp，理论支持 1000+ 网站。实际情况：

  • Works great: YouTube — videos, Shorts & playlists (no login needed).
  • 稳定可用：YouTube —— 视频、Shorts、播放列表（无需登录）。
  • Usually works for public posts: X/Twitter, Vimeo, SoundCloud, Dailymotion, Reddit, and more.
  • 多数可用（公开内容）：X/Twitter、Vimeo、SoundCloud、Dailymotion、Reddit 等。
⚠️ Honest limit: sites needing a login (private Instagram/Facebook, Douyin, Bilibili) won’t work without extra cookie setup this tool doesn’t include yet. ⚠️ 如实说明：需要登录的网站（私密 Instagram/Facebook、抖音、哔哩哔哩）需要额外的 Cookie 配置，本工具暂未内置，故无法下载。

6 Want to build your own?想自己做一个？

No magic here — it’s a thin script around two great open-source tools. The recipe: 其实没有黑科技 —— 它只是包在两个优秀开源工具外的一层薄脚本。配方如下：

⬇️
1. yt-dlp = the engine1. yt-dlp = 下载引擎 Does all the actual downloading from 1000+ sites. 负责从 1000+ 网站真正下载内容。
🔧
2. ffmpeg = merge/convert2. ffmpeg = 合并/转码 Merges video+audio, makes m4a, converts subs to .srt. 合并音视频、转 m4a、字幕转 .srt。
📋
3. A menu wrapper3. 一个菜单脚本 A PowerShell loop that builds the right yt-dlp flags for you. PowerShell 循环，替你拼好 yt-dlp 的参数。
📦
4. setup.bat4. setup.bat Auto-fetches the two binaries so users install nothing. 自动下载这两个程序，用户无需手动安装。

Try the core in 30 seconds

30 秒体验核心功能

Everything the app does starts from one command: 整个工具的起点就是这一条命令： 

# best-quality video
yt-dlp "https://youtu.be/VIDEO_ID"

# audio only
yt-dlp -x --audio-format m4a "URL"

# subtitles only (en + 中文)
yt-dlp --skip-download --write-subs --sub-langs en,zh-Hans "URL"

Wrap those in a script with a menu, and you’ve rebuilt ytget. The full source is on GitHub — fork it as your starting point. 把这些命令包进一个带菜单的脚本，就重建了 ytget。完整源码在 GitHub —— Fork 它作为你的起点。

Tips小贴士

  • File won’t play? Install VLC — it plays everything.
  • 文件无法播放？安装 VLC，支持所有格式。
  • Keep yt-dlp updated (menu option 4) — YouTube changes often; updates fix breakage.
  • 保持 yt-dlp 更新（菜单选项 4）—— YouTube 经常变动，更新可修复问题。
  • Shorts download like normal videos — just paste the /shorts/ URL.
  • Shorts 与普通视频相同，直接粘贴 /shorts/ 链接。
